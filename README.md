# 📊 Sales Forecasting Project

## 📌 Overview
This project focuses on forecasting future sales using historical retail data. The goal is to help businesses make better decisions regarding inventory, demand planning, and resource allocation.

---

## 📂 Dataset
- Superstore sales dataset (real-world retail data)
- Data was aggregated by date to create a time-series dataset

---

## ⚙️ Methodology
- Data cleaning and preprocessing
- Time-based feature engineering (year, month, day, etc.)
- Forecasting using:
  - Linear Regression
  - ARIMA (time-series model)

---

## 📊 Visualizations
- Sales trend over time
- Monthly and weekly patterns
- Forecast vs actual comparison
- Future sales prediction (next 30 days)
- Residual and error analysis

---

## 📈 Model Evaluation
- MAE (Mean Absolute Error)
- RMSE (Root Mean Square Error)

---

## 🏪 Business Insights
- Helps in inventory planning
- Supports demand forecasting
- Assists in financial planning
- Improves staffing decisions

---

## 🎯 Conclusion
The model successfully predicts future sales trends and provides actionable insights for business decision-making.
